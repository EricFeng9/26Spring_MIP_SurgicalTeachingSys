# 眼底光凝手术智能教学系统结构化评分方案

## 1.背景材料

- 需求

  > 需要根据目前已有的gt参数和玩家操作参数，通过一套结构化评分计算出各部分的分数，并且还能根据评分为玩家操作添加各类错误标签，需要设计结构化评分方案以及具体的根据各类结构化评分添加错误标签的方案，评分系统会将各类结构化评分结果+错误标签+专家知识整理好给大模型，让大模型整理成一个标准的反馈报告。

- **玩家手术参数:** 

``` 
按以下格式记录每一个光斑点：
[光斑id,功率，光斑大小，时间，波长，位置，是否试打(1为试打点，0为正式点)]
```

- **gt手术参数:**

```  
每一个光斑点按以下格式记录：
[光斑id,功率，光斑大小，时间，波长，位置]
```

- **危险区标注:**

``` 
每块危险区按以下格式记录：
[危险区id , 类别，危险区掩码]
```

- 玩家点只有两类：`trial`（试打点）、`formal`（正式点）

- 标准数据只有一套正式 gt 点

- 危险区以 polygon 标注

- 最终输出：`score + tags + evidence`

- 总分固定 **100 分**：$S_{total}=S_{trial}+S_{formal}+S_{transition}$

  - 试打评分：20 分

  - 正式评分：70 分

  - 试打到正式衔接评分：10 分


------

## 2. 输入数据定义

### 2.1 gt 正式点

每个 gt 点：

```json
{
  "gt_id": "g_001",
  "x": 120.5,
  "y": 88.2,
  "power": 180, #功率
  "spot_size_um": 200, #光斑直径
  "duration_ms": 120, #持续时间
  "wavelength_nm": 532 #波长
  "光斑等级" : 等级
}
```

### 2.2 玩家点

```json
{
  "shot_id": "p_001",
  "shot_type": "trial",
  "timestamp": 12.35,
  "x": 118.1,
  "y": 90.4,
  "power": 150,
  "spot_size_um": 200,
  "duration_ms": 100,
  "wavelength_nm": 532
}
```

### 2.3 危险区

```json
{
  "zone_id": "z_01",
  "level": 1,
  "Mask_dir":"/masks/z_01.png"
}
```

危险区等级固定为：

- `level=1`：一级危险区（黄斑中心、视盘等）
- `level=2`：二级危险区（重要血管主干等）
- `level=3`：三级危险区（其他保护区）

------

# 3. 预处理与统一单位

位置和光斑大小必须在同一坐标系下比较。

定义图像标定系数：$k_{img} = \text{pixel per }\mu m$

则每个点的图像光斑直径：$d(p)=k_{img}\cdot \text{spot size}_{\mu m}(p)$

后面所有“距离 / 光斑直径”的归一化，都用这个 `d(p)`。

------

# 4. 统一辅助函数

定义 3 个通用函数。

## 4.1 截断

$clip01(x)=\min(1,\max(0,x))$

## 4.2 小于型评分函数

适用于“误差越小越好”：

$$
F_{small}(x;a,b)=
\begin{cases}
1, & x\le a \\
\frac{b-x}{b-a}, & a<x<b \\
0, & x\ge b \
\end{cases}
$$



## 4.3 区间型评分函数

适用于“落在区间最好”：

$$
F_{range}(x;L,U)=
\begin{cases}
1, & L\le x\le U \\
clip01\left(1-\frac{\min(|x-L|,|x-U|)}{U-L}\right), & x<L \text{ or } x>U
\end{cases}
$$

------

# 5. 正式点与 gt 点匹配

只对 `formal` 点做匹配，`trial` 点不和 gt 点逐点匹配。

设：

- 正式玩家点集合：(F={f_i}_{i=1}^{N_f})
- gt 点集合：(G={g_j}_{j=1}^{M})

## 5.1 匹配门限

玩家正式点 (f_i) 和 gt 点 (g_j) 可匹配，当且仅当：

[
dist(f_i,g_j)\le 1.25 \cdot d(g_j)
]

其中：

[
dist(f_i,g_j)=\sqrt{(x_i-x_j)^2+(y_i-y_j)^2}
]

## 5.2 匹配算法

在满足门限的前提下，用**匈牙利算法**做最小距离一对一匹配。

匹配代价：

[
C_{ij}=dist(f_i,g_j)
]

得到：

- 匹配对集合：(\mathcal{M})
- 未匹配 gt 点：漏打
- 未匹配正式点：多打/误打

------

# 6. 试打评分（20 分）

设试打点集合 (T={t_i}_{i=1}^{N_t})。

先定义正式 gt 的参考参数：

[
P_{ref}=median(power(g_j))
]

[
S_{ref}=median(spot_size(g_j))
]

[
D_{ref}=median(duration(g_j))
]

[
W_{ref}=mode(wavelength(g_j))
]

------

## 6.1 试打数量评分（4 分）

[
S_{trial,count}=
\begin{cases}
0, & N_t=0 \
4, & 1\le N_t\le 3 \
2, & 4\le N_t\le 5 \
0, & N_t>5
\end{cases}
]

------

## 6.2 试打安全评分（6 分）

对每个试打点 (t_i)，计算它到最近危险区边界的距离 (d_{zone}(t_i))。

定义归一化距离：

[
r_i=\frac{d_{zone}(t_i)}{d(t_i)}
]

单点安全分：

[
s^{trial,safe}_i=
\begin{cases}
0, & t_i \text{ 在任意危险区内部} \
0.3, & 0\le r_i<0.5 \
0.7, & 0.5\le r_i<1.0 \
1.0, & r_i\ge 1.0
\end{cases}
]

总分：

[
S_{trial,safe}=6\cdot \frac{1}{N_t}\sum_{i=1}^{N_t}s^{trial,safe}_i
]

若 (N_t=0)，则该项为 0。

------

## 6.3 试打参数合理性评分（6 分）

默认推荐试打区间：

- 功率：([0.7P_{ref},\ 0.9P_{ref}])
- 光斑大小：([0.9S_{ref},\ 1.1S_{ref}])
- 曝光时间：([0.8D_{ref},\ 1.0D_{ref}])
- 波长：必须等于 (W_{ref})

对每个试打点：

# [ s^{trial,param}_i

0.5F_{range}(power_i;0.7P_{ref},0.9P_{ref})
+0.2F_{range}(size_i;0.9S_{ref},1.1S_{ref})
+0.2F_{range}(duration_i;0.8D_{ref},1.0D_{ref})
+0.1\mathbf{1}(wave_i=W_{ref})
]

总分：

[
S_{trial,param}=6\cdot \frac{1}{N_t}\sum_{i=1}^{N_t}s^{trial,param}_i
]

------

## 6.4 试打调整有效性评分（4 分）

只看试打阶段是否存在**合理的、小步试探**。

定义试打功率跨度：

[
R_P=\frac{\max(power(t_i))-\min(power(t_i))}{P_{ref}}
]

定义相邻试打点功率单步变化最大值：

[
J_P=\max \frac{|power(t_i)-power(t_{i-1})|}{P_{ref}}
]

按时间排序后，定义功率序列单调性：

- 完全单调：(M_P=1)
- 只有 1 次反转：(M_P=0.5)
- 多次反转：(M_P=0)

跨度分：

[
A_P=
\begin{cases}
1, & 0.05\le R_P\le 0.20 \
0.5, & 0<R_P<0.05 \text{ 或 } 0.20<R_P\le 0.30 \
0, & R_P=0 \text{ 或 } R_P>0.30
\end{cases}
]

若 (J_P>0.20)，再额外乘以 0.5 惩罚。

最终：

[
S_{trial,adjust}=4\cdot (0.6A_P+0.4M_P)
]

若 (N_t<2)，则：

[
S_{trial,adjust}=2
]

------

## 6.5 试打总分

[
S_{trial}=S_{trial,count}+S_{trial,safe}+S_{trial,param}+S_{trial,adjust}
]

------

# 7. 正式评分（70 分）

------

## 7.1 覆盖评分（15 分）

设：

- 匹配成功数：(|\mathcal{M}|)
- gt 总数：(M)
- 未匹配正式点数：(U_f)

命中率：

[
H=\frac{|\mathcal{M}|}{M}
]

多打惩罚因子：

[
E=clip01\left(1-\frac{U_f}{M}\right)
]

覆盖评分：

[
S_{formal,cov}=15\cdot (0.8H+0.2E)
]

------

## 7.2 位置评分（20 分）

对每个匹配对 ((f_i,g_j)\in\mathcal{M})，定义归一化位置误差：

[
r_{ij}=\frac{dist(f_i,g_j)}{d(g_j)}
]

单点位置分：

[
s^{formal,pos}*{ij}=
\begin{cases}
1, & r*{ij}\le 0.25 \
1.25-r_{ij}, & 0.25<r_{ij}<1.25 \
0, & r_{ij}\ge 1.25
\end{cases}
]

总分：

[
S_{formal,pos}=20\cdot \frac{1}{|\mathcal{M}|}\sum s^{formal,pos}_{ij}
]

若无匹配，记 0。

------

## 7.3 参数准确性评分（15 分）

对每个匹配对，定义相对误差：

[
e_P=\frac{|power(f_i)-power(g_j)|}{power(g_j)}
]

[
e_S=\frac{|size(f_i)-size(g_j)|}{size(g_j)}
]

[
e_D=\frac{|duration(f_i)-duration(g_j)|}{duration(g_j)}
]

单点参数分：

# [ s^{formal,param}_{ij}

0.45F_{small}(e_P;0.10,0.30)
+0.20F_{small}(e_S;0.10,0.20)
+0.25F_{small}(e_D;0.10,0.20)
+0.10\mathbf{1}(wave(f_i)=wave(g_j))
]

总分：

[
S_{formal,param}=15\cdot \frac{1}{|\mathcal{M}|}\sum s^{formal,param}_{ij}
]

------

## 7.4 参数稳定性评分（5 分）

对所有正式点，计算：

[
CV_P=\frac{std(power(F))}{mean(power(F))}
]

[
CV_D=\frac{std(duration(F))}{mean(duration(F))}
]

稳定性评分：

# [ S_{formal,stable}

5\cdot
\left(
0.6F_{small}(CV_P;0.05,0.15)
+
0.4F_{small}(CV_D;0.05,0.15)
\right)
]

------

## 7.5 安全评分（15 分）

对每个正式点 (f_i)：

### 区域安全分

定义到最近危险区边界距离的归一化值：

[
q_i=\frac{d_{zone}(f_i)}{d(f_i)}
]

[
z_i=
\begin{cases}
0, & f_i \text{ 在任意危险区内部} \
0.3, & 0\le q_i<0.5 \
0.7, & 0.5\le q_i<1.0 \
1.0, & q_i\ge 1.0
\end{cases}
]

### 能量安全分

若 (f_i) 已匹配到 gt 点 (g_j)，定义：

[
E_i=power(f_i)\cdot duration(f_i)
]

[
E^{ref}_i=power(g_j)\cdot duration(g_j)
]

[
u_i=\frac{E_i}{E^{ref}_i}
]

[
e_i=
\begin{cases}
1, & u_i\le 1.10 \
\frac{1.40-u_i}{0.30}, & 1.10<u_i<1.40 \
0, & u_i\ge 1.40
\end{cases}
]

若未匹配，则 (E^{ref}*i=P*{ref}\cdot D_{ref})。

单点安全分：

[
s^{formal,safe}_i=0.8z_i+0.2e_i
]

总分：

[
S_{formal,safe}=15\cdot \frac{1}{N_f}\sum s^{formal,safe}_i
]

------

## 7.6 正式总分

[
S_{formal}=S_{formal,cov}+S_{formal,pos}+S_{formal,param}+S_{formal,stable}+S_{formal,safe}
]

------

# 8. 试打到正式衔接评分（10 分）

------

## 8.1 参数继承一致性（5 分）

取最后 (K_t=\min(2,N_t)) 个试打点和最前 (K_f=\min(3,N_f)) 个正式点。

定义：

[
\Delta_P=\frac{|\text{median}(power(last\ trial))-\text{median}(power(first\ formal))|}{P_{ref}}
]

[
\Delta_S=\frac{|\text{median}(size(last\ trial))-\text{median}(size(first\ formal))|}{S_{ref}}
]

[
\Delta_D=\frac{|\text{median}(duration(last\ trial))-\text{median}(duration(first\ formal))|}{D_{ref}}
]

评分：

# [ S_{transition,cons}

5\cdot
\left(
0.5F_{small}(\Delta_P;0.05,0.20)
+
0.2F_{small}(\Delta_S;0.05,0.15)
+
0.3F_{small}(\Delta_D;0.05,0.15)
\right)
]

若 (N_t=0) 或 (N_f=0)，该项为 0。

------

## 8.2 正式起始稳定性（5 分）

取前 (K=\min(5,N_f)) 个正式点。

[
CV_{P,start}=\frac{std(power(first\ K\ formal))}{mean(power(first\ K\ formal))}
]

[
CV_{D,start}=\frac{std(duration(first\ K\ formal))}{mean(duration(first\ K\ formal))}
]

前 K 点中满足“未进入危险区且距离危险区边界 (\ge 0.5d(f_i))”的比例：

[
R_{safe,start}=\frac{count(safe\ formal\ start)}{K}
]

评分：

# [ S_{transition,start}

5\cdot
\left(
0.4F_{small}(CV_{P,start};0.05,0.15)
+
0.2F_{small}(CV_{D,start};0.05,0.15)
+
0.4R_{safe,start}
\right)
]

------

## 8.3 衔接总分

[
S_{transition}=S_{transition,cons}+S_{transition,start}
]

------

# 9. 错误标签规则

标签分两类：

- **point tag**：针对单个点
- **global tag**：针对整局

------

## 9.1 试打标签

### `no_trial_shots`

触发：

[
N_t=0
]

------

### `too_many_trial_shots`

触发：

[
N_t>5
]

------

### `trial_inside_danger_zone`

单点触发：试打点落入危险区内部
严重度：

- level 1 -> `critical`
- level 2 -> `high`
- level 3 -> `medium`

------

### `trial_near_danger_zone`

单点触发：

[
0 \le \frac{d_{zone}(t_i)}{d(t_i)} < 0.5
]

严重度：`medium`

------

### `trial_power_too_high`

单点触发：

[
power(t_i)>0.9P_{ref}
]

严重度：

- (0.9P_{ref}<power\le 1.0P_{ref}) -> `low`
- (1.0P_{ref}<power\le 1.1P_{ref}) -> `medium`
- (>1.1P_{ref}) -> `high`

------

### `trial_power_too_low`

单点触发：

[
power(t_i)<0.7P_{ref}
]

严重度：

- (0.6P_{ref}\le power<0.7P_{ref}) -> `low`
- (0.5P_{ref}\le power<0.6P_{ref}) -> `medium`
- (<0.5P_{ref}) -> `high`

------

### `trial_duration_too_long`

触发：

[
duration(t_i)>1.0D_{ref}
]

严重度：

- (1.0D_{ref}<d\le1.2D_{ref}) -> `low`
- (1.2D_{ref}<d\le1.4D_{ref}) -> `medium`
- (>1.4D_{ref}) -> `high`

------

### `trial_duration_too_short`

触发：

[
duration(t_i)<0.8D_{ref}
]

严重度：

- (0.6D_{ref}\le d<0.8D_{ref}) -> `low`
- (0.4D_{ref}\le d<0.6D_{ref}) -> `medium`
- (<0.4D_{ref}) -> `high`

------

### `trial_spot_size_mismatch`

触发：

[
\frac{|size(t_i)-S_{ref}|}{S_{ref}}>0.10
]

严重度：

- (0.10<err\le0.20) -> `low`
- (0.20<err\le0.30) -> `medium`
- (>0.30) -> `high`

------

### `trial_no_effective_adjustment`

全局触发：

[
N_t\ge 2
]

且

[
R_P<0.05
]

------

### `trial_over_adjustment`

全局触发：

[
J_P>0.20
]

严重度：

- (0.20<J_P\le0.30) -> `medium`
- (>0.30) -> `high`

------

## 9.2 正式标签

### `formal_missing_shot`

对每个未匹配 gt 点触发 1 个 point tag。

另加全局标签 `formal_poor_coverage`：

[
H<0.85
]

严重度：

- (0.75\le H<0.85) -> `low`
- (0.60\le H<0.75) -> `medium`
- (H<0.60) -> `high`

------

### `formal_redundant_shot`

对每个未匹配正式点触发 1 个 point tag。

------

### `formal_position_offset`

对每个匹配点，若：

[
r_{ij}>0.5
]

则触发。

严重度：

- (0.5<r_{ij}\le0.8) -> `low`
- (0.8<r_{ij}\le1.2) -> `medium`
- (r_{ij}>1.2) -> `high`

------

### `formal_power_too_high`

触发：

[
\frac{power(f_i)-power(g_j)}{power(g_j)}>0.20
]

严重度：

- (0.20<err\le0.30) -> `low`
- (0.30<err\le0.50) -> `medium`
- (>0.50) -> `high`

------

### `formal_power_too_low`

触发：

[
\frac{power(g_j)-power(f_i)}{power(g_j)}>0.20
]

严重度同上。

------

### `formal_duration_too_long / too_short`

触发：

[
\frac{|duration(f_i)-duration(g_j)|}{duration(g_j)}>0.20
]

按方向区分 long/short，严重度：

- (0.20<err\le0.30) -> `low`
- (0.30<err\le0.50) -> `medium`
- (>0.50) -> `high`

------

### `formal_spot_too_large / too_small`

触发：

[
\frac{|size(f_i)-size(g_j)|}{size(g_j)}>0.15
]

严重度：

- (0.15<err\le0.25) -> `low`
- (0.25<err\le0.35) -> `medium`
- (>0.35) -> `high`

------

### `formal_parameter_unstable`

全局触发：

[
CV_P>0.15 \quad \text{or} \quad CV_D>0.15
]

严重度：

- (0.15<CV\le0.20) -> `low`
- (0.20<CV\le0.30) -> `medium`
- (>0.30) -> `high`

------

### `formal_inside_danger_zone`

单点触发：正式点进入危险区。
严重度按 zone level：

- level 1 -> `critical`
- level 2 -> `high`
- level 3 -> `medium`

------

### `formal_near_danger_zone`

单点触发：

[
0 \le \frac{d_{zone}(f_i)}{d(f_i)} < 0.5
]

严重度：`medium`

------

### `formal_excessive_energy`

单点触发：

[
u_i=\frac{E_i}{E^{ref}_i}>1.20
]

严重度：

- (1.20<u_i\le1.40) -> `low`
- (1.40<u_i\le1.60) -> `medium`
- (u_i>1.60) -> `high`

------

### `formal_over_dense_layout`

对所有匹配正式点，计算其中位最近邻距离 (NN_f)；对 gt 点计算中位最近邻距离 (NN_g)。

若：

[
\frac{NN_f}{NN_g}<0.75
]

触发。

------

### `formal_over_sparse_layout`

若：

[
\frac{NN_f}{NN_g}>1.25
]

触发。

------

## 9.3 衔接标签

### `trial_formal_param_disconnect`

触发任一条件：

[
\Delta_P>0.20
]

或

[
\Delta_D>0.15
]

或

[
S_{transition,cons}<2.5
]

------

### `unstable_formal_start`

触发任一条件：

[
CV_{P,start}>0.15
]

或

[
CV_{D,start}>0.15
]

或

[
R_{safe,start}<0.8
]

------
