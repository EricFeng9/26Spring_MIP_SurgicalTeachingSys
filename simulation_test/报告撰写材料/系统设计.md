| 字段名| 面板名称  | 类型| 建议取值/范围| 备注  |
|---------------------|--------------|--------------|------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| `mode`    | 单点/矩阵选择 | enum    | `single` / `pattern`  | 设备支持单点和图形扫描。([Lumenis](https://lumenis.com/vision/products/photocoagulators/array-pattern-scanning/))|
| `wavelength_nm`| 激光波长  | enum    | `532` / `577` / `659` | Vision One 三波长。([Lumenis](https://lumenis.com/vision/products/photocoagulators/vision-one/))  |
| `power_mw`| 治疗功率  | number  | 532nm：`50–2000 mW`；577nm：`50–1500 mW`；659nm：`50–800 mW`  | 建议按波长联动限制。([Lumenis](https://lumenis.com/vision/products/photocoagulators/vision-one/))  |
| `duration_ms`  | 曝光时间  | number / enum| **单点**：`10–3000 ms`；**矩阵**：公开 Array 操作手册可见 `10 / 20 / 30 ms`  | 单点范围来自 Vision One 官方规格；矩阵离散值来自公开 Array 操作手册。([Lumenis](https://lumenis.com/vision/products/photocoagulators/vision-one/)) |
| `pulse_mode`   | SPL / 重复模式 | enum    | `single_pulse` / `repeat`  | 官方规格写有 “single pulse” 与 interval 模式。([Lumenis](https://lumenis.com/vision/products/photocoagulators/vision-one/)) |
| `interval_s`   | 光斑重复/间隔  | number / enum| `0.05–1.0 s`；仅单点下显示   | 官方规格给出 treatment interval 0.05–1.0 s；公开手册说明非单点时此字段会切换成 `spacing`。([Lumenis](https://lumenis.com/vision/products/photocoagulators/vision-one/)) |
| `spot_size_um` | 光斑大小  | number  | **当前这类扫描面板建议**：单点 `50–1000 µm`，矩阵 `100–500 µm` | Array 官方页面给出单点/图形模式的光斑范围。([Lumenis](https://lumenis.com/vision/products/photocoagulators/array-pattern-scanning/)) |
| `shape`   | 矩阵形状  | enum    | `single` / `square` / `line` / `triangle` / `circle` / `quarter_circle` / `half_circle`| 七种形状。([3B Optic Instruments](https://3boptic.com/manuales/Lumenis%20Array%20LaserLink%20Operator%20Manual.pdf))  |
| `shape_param`  | 点数/几何参数  | struct  | `square`: 每边 `2–5` 点；`line`: 从单线 2 点到 5 行；`triangle`: `6` 或 `15` 点；`circle`: 单环；`quarter_circle` / `half_circle`: `1` 或 `3` 条弧 | 做模拟时建议拆成 shape-specific 参数。([Lumenis](https://lumenis.com/vision/products/photocoagulators/array-pattern-scanning/)) |
| `spacing_x_spot`    | 光斑间距  | enum / number| **圆形**：`0–3 × spot diameter`；**其他图形**：`0.25–3 × spot diameter`| 用“相对光斑直径倍数”表达最合适。([Lumenis](https://lumenis.com/vision/products/photocoagulators/array-pattern-scanning/)) |
| `fundus_lens`  | 接触镜选择 | enum    | 常用 fundus lens 列表| 系统会根据镜片倍率计算 retina 上实际光斑。公开资料确认有镜片菜单，但**未公开完整镜片表**。([Lumenis](https://lumenis.com/vision/products/photocoagulators/array-pattern-scanning/)) |
| `aiming_beam_level` | 瞄准光亮度 | enum / int   | 可调；**公开资料未给出具体档位数**   | 官方只确认 635nm、<1.0mW、可调强度。([Lumenis](https://lumenis.com/vision/products/photocoagulators/vision-one/))|
